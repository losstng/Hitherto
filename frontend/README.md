# Hitherto Frontend

Next.js application providing the web interface for Hitherto.

## Setup

```bash
npm install
```

Create a `.env.local` file with the backend URL:

```env
NEXT_PUBLIC_API_BASE_URL=http://localhost:8000
```

## Development

Start the development server:

```bash
npm run dev
```

## Testing

Run lint checks:

```bash
npm run lint
```
